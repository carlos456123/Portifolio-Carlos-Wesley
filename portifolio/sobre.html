Sobre mim

Trajetoria Academica

Graduando em Sistemas de Informação, na Universidade Pernambuco (início em 2024), com foco em unir excelência técnica à visão estratégica de negócios. Possuo experiência prática no desenvolvimento Backend com Java e Python, aplicando estruturas de dados, padrões de projeto e operações de CRUD integradas a bancos de dados SQL. Atualmente, amplio meu ecossistema de habilidades no Frontend, explorando HTML5, CSS3, JavaScript e jQuery para criar interfaces funcionais e intuitivas. Além do código, possuo sólida base em gestão organizacional, processos contábeis e sistemas de apoio à decisão, o que me permite entender a tecnologia como ferramenta de valor para o negócio. Busco evolução contínua e oportunidades para aplicar inovação na resolução de problemas reais.
