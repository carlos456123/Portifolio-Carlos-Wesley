##  Descrição do Projeto

Este projeto consiste no desenvolvimento de um site de portfólio pessoal com 5 páginas interligadas, com o objetivo de apresentar informações profissionais, habilidades, cursos e formas de contato do integrante.
O site foi desenvolvido utilizando HTML, CSS e JavaScript.
---

##  Estrutura do Site

* **index.html** → Página inicial com apresentação
* **sobre.html** → Informações pessoais e objetivos
* **habilidades.html** → Habilidades técnicas e comportamentais
* **cursos.html** → Cursos e formações
* **contato.html** → Formulário e informações de contato

---

## Tecnologias Utilizadas

* HTML5 (estrutura semântica)
* CSS3 (estilização e responsividade)
* JavaScript (interatividade e validações)

---

##  Como Executar o Projeto

1. Baixe ou clone este repositório
2. Extraia os arquivos (se estiverem compactados)
3. Abra a pasta do projeto
4. Clique duas vezes no arquivo: index.html

---

## ✨ Funcionalidades

* Menu de navegação responsivo
* Validação de formulário
* Exibição de habilidades com porcentagem
* Seção de ferramentas com interação
